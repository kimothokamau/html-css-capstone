REACT JS AFRICA EDITION 2021

This conference brings together a community of developers, programmers, students, enthusiasts who are passionate about React Java Script framework.

The goal is to advance knowledge of the attendees, discuss industry best practices, deep dive into new React features, showcase applications built as well as connect employers to potential employees looking for new opportunities
